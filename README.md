# Pesquisa_Me
 Neste projeto, construí um APP de Busca utilizando HTML5, Sass, Javascript e a API da Wikipedia.
 O design da página é 100% responsivo.
 
 
 # Pesquisa Me
 
 Acesse o site [aqui.](https://pesquisa-me.netlify.app/)
